# ONE-APP-CLI
CLI for [ONE](http://wufazhuce.com/)

![](http://wx3.sinaimg.cn/large/65ad7d5aly1g1cx1w1xnij210w0mse4t.jpg)

*Currently only supported on [iTerm >=3](https://www.iterm2.com/downloads.html).*
## Features
- Fetch today

## Install

```bash
$ npm i -g one-app-cli
```
## Usage

### Commands

```bash
$ one -v    # Show version
$ one today   # Show today info
```
