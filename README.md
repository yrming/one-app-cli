# ONE-APP-CLI
CLI for [ONE](http://wufazhuce.com/)

*Currently only supported on [iTerm >=3](https://www.iterm2.com/downloads.html).*
## Features
- Fetch today

## Install

```bash
$ npm i -g one-app-cli
```
## Usage

### Commands

```bash
$ one -v    # Show version
$ one today   # Show today info
```
